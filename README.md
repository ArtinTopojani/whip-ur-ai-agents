# Whip Cursor

> A tiny Windows tray app for whipping your screen when your AI agents need motivation.

## Install

Download the GitHub ZIP, extract it, then double-click:

```text
Install Whip Cursor.cmd
```

The installer builds the app, installs it for your Windows user account, creates shortcuts, and starts Whip Cursor.

If something needs to be downloaded first, it will ask before continuing.

## Run

After installing, double-click:

```text
Run Whip Cursor.cmd
```

The installer also creates a **Whip Cursor** Desktop shortcut.

## Controls

| Action | What it does |
| --- | --- |
| Left-click | Plays the whip animation at your cursor |
| F7 | Mutes or unmutes the whip sound |
| F8 | Arms or disarms the effect |
| Ctrl+Shift+Q | Quits the app |
| Right-click tray icon | Opens arm/disarm, mute SFX, and exit options |

Your clicks still pass through to whatever is underneath.

## Developer Run

To test quickly without installing, run:

```text
tools\Run Dev Build.cmd
```
