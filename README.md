# Whip Cursor

> A tiny Windows tray app for whipping your screen when your AI agents need motivation.

## Install

1. Download the project ZIP from GitHub.
2. Extract the ZIP.
3. Double-click:

```text
Run Whip Cursor.cmd
```

The script will build Whip Cursor, install it for your Windows user account, create a Desktop shortcut, and start the app.

If something needs to be downloaded first, the script will ask before opening the download page or continuing.

After the first install, use the **Whip Cursor** shortcut on your Desktop.

## Controls

| Action | What it does |
| --- | --- |
| Left-click | Plays the whip animation at your cursor |
| F8 | Arms or disarms the effect |
| Ctrl+Shift+Q | Quits the app |
| Right-click tray icon | Opens arm/disarm and exit options |

Your clicks still pass through to whatever is underneath.
